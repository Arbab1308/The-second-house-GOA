## Updating Images

### How to Replace an Image
1. Upload new image to Vercel Blob Storage
2. Copy new blob URL
3. Find component using old image (use grep)
4. Update URL in component
5. Test locally: `npm run dev`
6. Deploy: `git push`

### How to Add a New Image
1. If replacing one of 20 - update existing URL
2. If adding new section - update page.tsx import
3. Follow same blob URL pattern
4. Add alt text (required for SEO)
5. Use useInView hook for animations
